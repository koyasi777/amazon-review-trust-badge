Amazon Reviewer Trust Badge は、**レビュアー（投稿者）の過去の活動履歴を統計的に分析**し、信頼度を **S〜Dランク** のバッジとして表示するブラウザ拡張スクリプト（UserScript）です。  
サクラレビュー・組織票・提供品バイアスなど、**レビューの“書き手由来の偏り”** を素早く見抜くための補助ツールです。

さらにレビュー一覧を快適化するために、**無限スクロール** と **信頼度フィルタ** を追加します。  
「読む価値のあるレビュー」だけに集中できます。

---

## ✨ 主な機能

このスクリプトは、レビューの内容を“断定”するものではありません。  
**投稿者の行動パターンから「信頼できそうか／注意が必要か」を可視化**し、意思決定を早くします。

### 🏷️ 1. 信頼度スコアリング & バッジ表示（S〜D）
レビュアーの投稿履歴を解析し、信頼度を **S（最高）〜D（危険/低品質）** のランクで評価します。  
レビューの投稿者名の横に **バッジ** を表示します。

- **S / A:** 多様な評価分布、十分な情報量、写真投稿、継続的に「参考になった」票を獲得
- **C:** 可もなく不可もなく。偏りは少ないが決定打に欠ける
- **D:** 全件★5・短文量産・不自然な投票パターンなど、注意すべき兆候が多い

### 📊 2. 詳細分析レポート（クリックで即表示）
バッジをクリックすると、レビュアーの分析レポートがポップアップします。

- **評価分布グラフ:** 星の偏り（例：全件★5）が一目で分かる
- **統計:** 平均文字数、写真率、「参考になった」獲得状況 など
- **警告タグ:** サクラ疑い / Vine / 未購入 など、リスク要因をラベル化

### 🧲 3. 信頼度フィルタリング（“怪しいレビューを物理的に消す”）
レビュー一覧ページに **信頼度フィルタ** を追加します。

- 例：**B以上のみ表示** / **Dランク非表示** など
- ノイズを除去し、判断材料として有効なレビューに集中できます

### ♾️ 4. 無限スクロール（レビューを止めずに読める）
次ページを自動読み込みし、レビュー一覧を **シームレスに連続表示** します。  
「ページ移動 → 読み込み待ち」を消してテンポよく調査できます。

## 🏷️ 分析タグ一覧（意味と傾向）

スクリプトは、レビュアーの特徴をタグで分類します。

| タグ | ラベル | 意味と傾向 |
| --- | --- | --- |
| **Div** | 🛡️ 自然分布 | 星の分布が自然で極端な偏りが少ない（信頼できる傾向） |
| **Deep** | 📝 長文詳細 | 文字数が多く、具体性のあるレビューが多い |
| **Img** | 📷 写真あり | 実機確認の証拠になりやすい（信頼性上昇） |
| **Gold** | 🏅 高品質 | 「参考になった」票を継続的に獲得している |
| **All5** | 🚨 全件★5 | 盲目的高評価。業者/バイアスの可能性が上がる |
| **Thin** | 🍂 内容希薄 | 情報価値が低い短文が多い |
| **Swarm** | 🐝 組織票疑 | 内容に対して不自然に票が多い（操作の疑い） |
| **Gap** | 📉 品質乖離 | 特定商品だけ異常に高評価など、行動が不自然 |
| **Vine** | 🎁 Vine | 無償提供レビュー（好意的バイアスが入りやすい） |
| **Non** | ⚠️ 未購入 | 「Amazonで購入」なし。外部購入/架空の可能性 |

## 📥 インストール方法

このツールは **UserScript** です（PCブラウザ + 一部のモバイルブラウザで動作）。

1. **ユーザースクリプトマネージャーを導入**
   - [Violentmonkey](https://violentmonkey.github.io/)
   - [Tampermonkey](https://www.tampermonkey.net/)

2. **スクリプトをインストール**
   - 👉 [ここをクリックしてインストール](https://github.com/koyasi777/amazon-review-trust-badge/raw/main/amazon-review-trust-badge.user.js)

3. **Amazonの商品ページを開く**
   - 自動的に解析が始まり、投稿者名の横にバッジが表示されます

## 📖 使い方ガイド

### 基本
- **バッジを見る:** 投稿者名の横の **S〜D** を確認
- **詳細を見る:** バッジをクリック → 分析レポートを表示
- **レビューを絞る:** レビュー一覧ページで信頼度フィルタを選択
- **連続で読む:** 無限スクロールでページ移動なしに閲覧

### 典型的な使い方（おすすめ）
- まず **D / All5 / Thin / Non** が多いレビューを除外
- 残った中から **Div / Deep / Img** の比率が高いレビュアーのレビューを読む
- Vine は「悪」ではないが、**バイアス要因として分離して読む**（推奨）

## 🧠 信頼度判定の考え方（アルゴリズム概要）

本スクリプトは、テキストの感情分析などではなく、**行動履歴の統計特徴**を中心に評価します。

### 加点要素（+）
- **多様性（Div）:** 星のばらつきがあり、良い点・悪い点が混在
- **情報量（Deep）:** 平均文字数が多く、具体性がある
- **実績（Gold）:** 「参考になった」票を安定して獲得
- **実在証明（Img）:** 写真付きレビュー率が高い

### 減点要素（-）
- **盲信的評価（All5）:** 投稿の大半が★5（不自然な偏り）
- **内容の欠如（Thin）:** 一言レビューの量産
- **文脈の欠如（Non）:** 未購入レビュー（判断材料が薄い）
- **提供品バイアス（Vine）:** 提供品による好意的バイアス

> 重要：これは推定値です。**S=絶対安全 / D=絶対サクラ** の断定ではありません。  
> 人間の判断を速くするための「レーダー」として使ってください。
